6. Sentencias de Control de Flujo

Las sentencias de control de flujo permiten modificar la ejecución de un programa dependiendo de condiciones específicas. En JavaScript, estas incluyen if...else, switch, y bucles como for, while, y do...while.

6.1 if...else

La sentencia if...else ejecuta un bloque de código si una condición es verdadera y otro si es falsa. 


            // Cuando edad sea mayor o igual a 18, imprime "Mayor de edad".
            if (edad >= 18) { 
                console.log("Mayor de edad");
            //  Sino, imprime "Menor de edad".
            } else { 
                console.log("Menor de edad");
            }

6.2 switch...case

switch...case evalúa una expresión y ejecuta el bloque correspondiente al valor coincidente de los case. Si no coincide con ninguno, se ejecuta el bloque default.


            // Según elvalor de la variable dia, imprime un mensaje diferente.
            switch (dia) {
                case "lunes":
                    console.log("Hoy es lunes");
                    break;
                case "martes":
                    console.log("Hoy es martes");
                    break;
                // Si no coincide con ningún caso, imprime "Día desconocido".
                default:
                    console.log("Día desconocido");
            }

6.3 for

El bucle for ejecuta un bloque de código un número determinado de veces, ideal para iterar sobre arrays o realizar acciones repetitivas.


            // El bucle se ejecuta 5 veces, desde i = 0 hasta i = 4.
            for (let i = 0; i < 5; i++) {
                // Imprime los números del 0 al 4.
                console.log(i); 
            }

6.4 while y do...while

while ejecuta un bloque de código mientras una condición sea verdadera. do...while garantiza que el código se ejecute al menos una vez.


        let i = 0;
        while (i < 5) { // El bucle se ejecuta 5 veces, desde i = 0 hasta i = 4.
            // Imprime los números del 0 al 4. 
            console.log(i);
            // Incrementa en 1 la variable i.
            i++; 
        }
        

        let i = 0;
        // El bucle se ejecuta una vez al menos.
        do {
            // Imprime los números del 0 al 4. 
            console.log(i);
            // Incrementa en 1 la variable i.
            i++;
        // El bucle se ejecuta 5 veces, desde i = 0 hasta i = 4.
        } while (i < 5);

6.5 break y continue

La sentencia break sale de un bucle, mientras que continue salta a la siguiente iteración sin ejecutar el código restante en el bucle.


        for (let i = 0; i < 10; i++) {
            // Se sale del bucle cuando i es 5.
            if (i === 5) break;
            console.log(i);
        }
        

        for (let i = 0; i < 10; i++) {
            // No se imprime cuando i es 5 y continúa con la siguiente iteración.
            if (i === 5) continue;
            console.log(i);
        }
