8. OBJETOS

JavaScript es un lenguaje orientado a objetos donde todo puede representarse como un objeto, ya sea físico o conceptual. Los objetos se definen por propiedades (atributos) y métodos (acciones que pueden realizar). Esto permite estructurar mejor los programas, representando entidades del mundo real mediante un modelo de datos que facilita su manipulación.

8.1 Propiedades

Las propiedades son características de un objeto y pueden almacenar valores primitivos u otros objetos. Se accede a ellas mediante la notación punto objeto.propiedad o con corchetes objeto["propiedad"]. Además, JavaScript permite agregar, modificar o eliminar propiedades dinámicamente, lo que da flexibilidad a la estructura de los objetos.


            const coche = {
                marca: "Ferrari",
                modelo: "Testarossa",
                potencia: 400
            };
            console.log(coche.marca); // "Ferrari"

También pueden modificarse o agregarse dinámicamente:


            coche.color = "Rojo";
            delete coche.potencia;

8.2 Métodos

Los métodos son funciones asociadas a un objeto y definen su comportamiento. Un método puede utilizar la palabra clave this para acceder a otras propiedades dentro del mismo objeto, lo que permite una mejor encapsulación de la lógica.


            const coche = {
                marca: "Ferrari",
                modelo: "Testarossa",
                mostrarDatos() {
                    // Utiliza "this" para acceder a las propiedades del objeto
                    return `${this.marca} ${this.modelo}`;
                }
            };
            console.log(coche.mostrarDatos());

También pueden añadirse o eliminarse dinámicamente:


            coche.arrancar = function() {
                return "El coche ha arrancado.";
            };
            delete coche.mostrarDatos;

8.3 La Función Constructora

Permite crear múltiples objetos con una estructura común. Es útil cuando se necesita instanciar muchos objetos similares sin repetir código. Al utilizar una función constructora, se pueden definir propiedades y métodos comunes para los objetos creados con new.


            function Coche(marca, modelo, potencia) {
                this.marca = marca;
                this.modelo = modelo;
                this.potencia = potencia;
                this.mostrarDatos = function() {
                    return `${this.marca} ${this.modelo}`;
                };
            }
            const miCoche = new Coche("BMW", "320i", 140);
            console.log(miCoche.mostrarDatos());

8.4 Mutabilidad

Los objetos en JavaScript son mutables, lo que permite modificar sus propiedades sin cambiar su identidad. Esto significa que al asignar un objeto a otra variable, ambas hacen referencia al mismo objeto en memoria.


            const miCoche = new Coche("Audi", "A4", 180);
            miCoche.potencia = 200;  // Se modifica la propiedad

Si se asigna un objeto a otra variable, ambas referencias apuntan al mismo objeto.


            const otroCoche = miCoche;
            otroCoche.potencia = 250;
            console.log(miCoche.potencia); // 250

8.5 Objetos Globales

JavaScript proporciona varios objetos globales que están integrados en el lenguaje, como Array, String, Date y Math. Estos objetos facilitan operaciones comunes y optimizan el manejo de datos.

8.5.1 El objeto Array

Los arrays permiten almacenar listas de valores ordenados y acceder a ellos mediante índices. Además, JavaScript ofrece una gran cantidad de métodos para manipular arrays de manera eficiente.


            let numeros = [1, 2, 3, 4];
            numeros.push(5);
            numeros.pop();

8.5.1.1 Acceso a los elementos de un array

Los elementos de un array se acceden mediante su índice, comenzando desde 0. También se pueden modificar de forma directa usando la misma notación.


            let numeros = [1, 2, 3, 4];
    
            // Accede al primer elemento
            console.log(numeros[0]); // 1
    
            // Modifica el primer elemento
            numeros[0] = 10;

8.5.1.2 Uso de arrays como colas o stacks

Un array puede actuar como una estructura de datos de tipo pila (LIFO) o cola (FIFO), usando métodos como push(), pop(), shift() y unshift().


            // Agrega al inicio
            numeros.unshift(0);
    
            // Elimina el primer elemento
            numeros.shift(); 
    
            // Agrega al final
            numeros.push(0);
    
            // Elimina el último elemento
            numeros.pop();

8.5.1.3 Métodos de unión, división y transformación de arrays

JavaScript permite manipular arrays mediante métodos como concat() para unir arrays, slice() para extraer porciones y map() para transformar elementos.


            let pares = [2, 4, 6];
            let impares = [1, 3, 5];
    
            // Une ambos arrays
            let numeros = pares.concat(impares);
    
            // Extrae elementos del índice 1 al 3
            let parte = numeros.slice(1, 4);
    
            // Transforma cada elemento al cuadrado
            let cuadrados = numeros.map(x => x * x);

8.5.2.1 Acceso a los caracteres de una cadena

Las cadenas de texto son indexadas como los arrays, lo que permite acceder a caracteres individuales utilizando índices.


            let texto = "JavaScript";
    
            // Accede al primer carácter
            console.log(texto.charAt(0)); // "J"
            
            // Accede al segundo carácter usando corchetes
            console.log(texto[1]); // "a"

8.5.3.1 Métodos de manejo de fechas

El objeto Date incluye métodos para extraer información específica como el año, mes o día, así como para realizar cálculos con fechas.


            // Crea un objeto Date con la fecha y hora actuales
            let fecha = new Date();
    
            // Año actual
            console.log(fecha.getFullYear());
    
            // Mes actual (0-11)
            console.log(fecha.getMonth());

8.5.4 El objeto Math

El objeto Math proporciona funciones matemáticas avanzadas, como generación de números aleatorios y cálculo de potencias.


            // Genera un número aleatorio entre 0 y 1
            console.log(Math.random());
    
            // Calcula 2 elevado a la 3ª potencia
            console.log(Math.pow(2, 3));
