7. FUNCIONES

Las funciones son bloques de código reutilizables que realizan tareas específicas, ayudando a la modularidad y reutilización del código. Se definen con la palabra clave function, seguida del nombre de la función y sus argumentos.


            function nombreFuncion(argumento1, argumento2) {
                // Código de la función
            }

7.1 Invocación de una función

Para invocar una función, se utiliza su nombre seguido de paréntesis, dentro de los cuales se pasan los argumentos.


            function suma(a, b) {
                return a + b;
            }
            
            // Invocación con argumentos:
            let resultado = suma(3, 5); // resultado será 8

Si no se conoce el número de argumentos, se puede usar la sintaxis de "rest parameters" (...) para recibir una lista de argumentos.


            function sumar(...listaArgumentos) {
                let suma = 0;
                for (let i = 0; i < listaArgumentos.length; i++) {
                    suma += listaArgumentos[i];
                }
                return suma;
            }
            
            let resultado = sumar(1, 2, 3); // resultado será 6

7.2 El tipo de datos Function

Las funciones en JavaScript son valores de tipo Function, lo que significa que pueden ser asignadas a variables y tratadas como cualquier otro valor.


            // Definir una función y asignarla a una variable de tipo Function.
            let miSaludo = function(nombre) {
                return "Hola, " + nombre + "!";
            };
    
            // Invocar la función a través de la variable de tipo Function.
            let mensaje = miSaludo("Juan"); // mensaje será "Hola, Juan!"

7.3 Alcance de las variables

Las variables en JavaScript pueden tener alcance global o local. Las variables globales son accesibles desde cualquier parte del programa, mientras que las locales solo son accesibles dentro de la función donde se definen.


            // Variable global
            let variableGlobal = "Soy global"; 
            
            function miFuncion() {
                // Variable local
                let variableLocal = "Soy local";
                
                // Accede a la variable global
                console.log(variableGlobal);
    
                // Accede a la variable local
                console.log(variableLocal); 
            }

Las variables locales se destruyen cuando la función termina, mientras que las globales siguen accesibles en otras partes del código.
