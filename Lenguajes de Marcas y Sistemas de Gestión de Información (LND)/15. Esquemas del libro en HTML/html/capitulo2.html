2. SINTAXIS BÁSICA

JavaScript sigue la especificación ECMAScript, y en esta obra se basa principalmente en la versión ES6 (ECMAScript 2015). Como cualquier otro lenguaje, JavaScript se compone de sentencias, que son instrucciones que indican qué hacer en cada momento. Estas sentencias se separan por un punto y coma (;), aunque el salto de línea también puede indicar el final de una sentencia. Es recomendable siempre usar el punto y coma para evitar errores inesperados.

Las sentencias en JavaScript están formadas por:

  • Valores: Son datos de diferentes tipos (booleanos, números, textos, etc.).
  • Variables: Almacenan valores. Se pueden declarar con palabras clave como let y const.
  • Operadores: Permiten realizar operaciones, como la suma (+) entre valores.
  • Palabras clave: Definen las acciones que el lenguaje debe realizar, como let para declarar variables.

Comentarios

Los comentarios son útiles para explicar el código y facilitar su mantenimiento. Pueden ser de una línea o de múltiples líneas:


            // Esto es un comentario de una línea
            
            /* Este es un comentario 
            de múltiples líneas */

Los comentarios no se ejecutan, lo que permite desactivar partes del código temporalmente para realizar pruebas o para aclarar su propósito.
