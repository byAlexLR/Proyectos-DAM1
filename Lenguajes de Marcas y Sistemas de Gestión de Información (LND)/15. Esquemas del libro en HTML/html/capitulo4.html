4. TIPOS DE DATOS PRIMITIVOS EN JAVASCRIPT

JavaScript tiene 7 tipos de datos primitivos, que son los bloques básicos con los que se trabaja en la programación. Estos tipos de datos determinan el conjunto de valores que una variable puede almacenar y las operaciones que se pueden realizar con ellos.

En muchos lenguajes de programación, se requiere definir el tipo de una variable al declararla. Sin embargo, en JavaScript, gracias al tipado dinámico, no es necesario especificar el tipo de una variable de antemano.

Los tipos de datos en JavaScript se dividen en primitivos (básicos) y compuestos (formados por varios tipos primitivos). Aquí nos enfocamos en los tipos primitivos, que son:

  • Boolean: Representa los valores true y false.
  • Number: Cualquier número, ya sea entero o decimal.
  • BigInt: Para representar números enteros que superan el rango de Number.
  • String: Cadenas de caracteres, que pueden ir entre comillas simples o dobles.
  • Null: Representa la ausencia intencionada de valor, solo puede ser null.
  • Undefined: Indica que una variable ha sido declarada, pero no se le ha asignado un valor, y solo puede tener el valor undefined.

Además, existe el tipo Symbol, que no se aborda en este contexto.

4.1 Boolean

El tipo Boolean tiene solo dos posibles valores: true y false. Se utiliza comúnmente en expresiones de comparación o lógica y en estructuras de control condicional, como if...else o switch. 


            let esMayor = true;

4.2 Number

En JavaScript, todos los números (enteros y decimales) pertenecen al tipo Number. Estos números se almacenan como valores de coma flotante de doble precisión (IEEE 754), lo que significa que ocupan 64 bits de memoria. Aunque este formato ofrece una precisión bastante alta, existen límites que pueden provocar redondeo en ciertos casos. Para trabajar con números más grandes sin perder precisión, se puede usar BigInt. 


            // Entero
            let edad = 30;
            
            // Decimal
            let precio = 99.99;
            
            // BigInt
            let numeroGrande = 1234567890123456789012345678901234567890n;

4.3 String

El tipo String se usa para representar secuencias de caracteres. Estas cadenas pueden estar delimitadas por comillas simples o dobles, y pueden incluir caracteres alfanuméricos o secuencias de escape como:

  • \b (retroceso)
  • \n (salto de línea)
  • \r (retorno de carro)
  • \t (tabulador)

            // Cadena con comillas dobles
            let nombre = "Juan";
    
            // Cadena con comillas simples
            let apellido = 'Pérez';
    
            // Cadena con secuencias de escape
            let mensaje = "Hola, \n¿cómo estás?";
    
            // Cadena con comillas simples dentro de comillas dobles y viceversa
            let frase = 'Ella dijo: "Hola"';

4.4 Null y Undefined

El valor null indica la ausencia intencionada de valor, mientras que undefined se refiere a una variable que ha sido declarada pero no ha sido inicializada. Aunque ambos pueden indicar "vacío", tienen significados distintos: null es un valor explícito, mientras que undefined es el valor por defecto en variables no inicializadas.


            // Variable con valor null
            let valorNulo = null;
    
            // Variable sin inicializar (valor undefined)
            let valorIndefinido;

4.5 Conversión entre Tipos de Datos

JavaScript permite la conversión entre diferentes tipos de datos primitivos mediante funciones globales. A continuación, algunos ejemplos:

4.5.1 Conversión a Números

Convertir una cadena a número con la función Number():


            let numero = Number("42"); // Devuelve 42
    
            // Otra forma de convertir a número es usando parseInt
            let numeroEntero = parseInt("42"); // Devuelve 42
    
            // O usando parseFloat para números decimales
            let numeroDecimal = parseFloat("3.14"); // Devuelve 3.14

4.5.2 Conversión a Cadenas

Convertir un número a cadena con la función String():


            let texto = String(123); // Devuelve "123"
    
            //Otra forma de convertir a cadena es usando toFixed
            let numero = 3.14159;
            let textoDecimal = numero.toFixed(2); // Devuelve "3.14"
    
            //Otra forma de convertir a cadena es usando toExponential
            let textoExponencial = numero.toExponential(2); // Devuelve "3.14e+0"
    
            //Otra forma de convertir a cadena es usando toPrecision
            let textoPrecision = numero.toPrecision(2); // Devuelve "3.1"

4.5.3 Conversión a Booleanos

Convertir valores a booleanos con la función Boolean(). Ejemplos:


            Boolean(0); // Devuelve false
            Boolean(1); // Devuelve true
