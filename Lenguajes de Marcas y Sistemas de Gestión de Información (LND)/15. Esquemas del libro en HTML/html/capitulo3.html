3. VARIABLES Y CONSTANTES

Las variables son espacios de memoria que almacenan información que puede cambiar durante la ejecución del programa, mientras que las constantes mantienen un valor fijo e inalterable una vez asignado.

3.1 Variables

Las variables en JavaScript se declaran con let o var, siendo let preferido para el ámbito de bloque. Aunque var es compatible con versiones anteriores, let es más seguro para evitar errores. Las variables pueden declararse y asignarse en una sola línea:


            let nombre = "Juan";
            var edad = 30;

Es recomendable usar la nomenclatura camelCase para nombrar variables compuestas, como nombreLargoDeVariable.

3.2 Constantes

Las constantes se declaran con const y su valor no puede cambiar después de la asignación inicial. Es obligatorio asignar un valor al declarar una constante:


            const PI = 3.1416;

Intentar modificar el valor de una constante como PI generará un error.
